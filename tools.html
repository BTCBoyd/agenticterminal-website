About This Directory

We don't build tools—we review, rank, and guide. This directory curates the best Lightning wallets, payment protocols, and infrastructure for AI agents with honest, detailed evaluations.

Bitcoin-native AND competing rails: We track L402 (Lightning), x402 (stablecoin), and traditional payment infrastructure. If Coinbase Agentic Wallets outperform Lightning-native solutions in adoption or UX, we document that honestly and analyze why. Orange accent = Bitcoin-native. Blue accent = Crypto-generic/competing rails.

Evaluation criteria: Agent-friendly API | Payment speed | Transaction costs | Developer experience | Documentation quality | Production readiness | Adoption metrics

Submission Guidelines

Building tools for AI agent payments? We'd love to review your work. Email submissions to tools@agenticterminal.ai with:

  • › Documentation link (API docs, integration guides)
  • › GitHub repo (if open source)
  • › Production examples (agents using your tool in the wild)
  • › Transaction metrics (volume, count, if public)

We evaluate all submissions based on merit, not ideology. If your tool works well for agents, we'll say so—even if it's not Bitcoin-native.
